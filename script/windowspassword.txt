---Change a Windows Password---
STEP1:(
Plug in (SR:USB)
boot to SR:USB
open file explorer navigate to system32
search for "sethc.exe" rename it to "sethc1.exe"
search for "cmd.exe" copy and paste it
rename your new pasted "cmd1.exe" to "sethc.exe")

reboot to windows

STEP2:(
in the windows login screen spam your right shift key
cmd will open
type "net user" this will give you a list of the users on the windows system
now type "net user yourusername *"
now it will ask you to type in the new password for the user
make sure to type it twice as the same thing or else it wont work
you have successfully changed the password on the user)
